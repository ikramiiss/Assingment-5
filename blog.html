Important FAQ

what is the Document object Mobel(DoM)?
The DOM is the bridge between HTML (structure) and JavaScript (behavior). It lets scripts read and update the content, structure, and style of a webpage.
How do you select on eiement from the DOM?
Selecting elements from the Document Object Model (DOM) in JavaScript involves using various methods provided by the document object. These methods allow you to retrieve references to HTML elements based on their attributes or position within the document structure.
what is event delegation in the context of the Dom,and why is it useful?
Event delegation is a JavaScript technique where a single event listener on a parent element handles events for all its descendants, leveraging the event bubbling mechanism. It is useful because it improves performance and reduces memory usage by avoiding numerous individual listeners, simplifies code by centralizing event handling, and efficiently manages dynamically added elements without needing to attach new listeners.
How do you manipulate on element's atributes and styles using the DOM?
Manipulating an element's attributes and styles using the Document Object Model (DOM) in JavaScript involves selecting the element and then utilizing various properties and methods.
